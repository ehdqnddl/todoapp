# 할 일 관리 (Todoapp)

프레임워크나 빌드 도구 없이 순수 HTML/CSS/JS로 만든 할 일 관리 웹앱입니다. PWA로 핸드폰 홈 화면에 설치해 오프라인에서도 쓸 수 있습니다.

**실사용 주소**: https://ehdqnddl.github.io/todoapp/

## 사용 방법

- **추가**: 입력창에 텍스트 입력 → 카테고리 선택 → Enter 또는 "추가" 버튼
- **완료 체크**: 항목 좌측 체크박스 클릭 (완료 항목은 취소선 표시 후 목록 하단으로 자동 정렬)
- **수정**: 할 일 텍스트 클릭 또는 더블클릭 → 인라인 편집 모드 → Enter 또는 다른 곳 클릭(포커스 아웃)으로 저장
- **삭제**: 항목 우측 × 버튼 클릭 → 화면 하단에 5초간 "취소" 스낵바가 뜨며, 클릭하면 복구됩니다
- **카테고리 변경**: 항목의 색상 태그(드롭다운)를 클릭해 업무/개인/공부 중 다른 카테고리로 즉시 변경할 수 있습니다
- **카테고리 필터**: 상단 탭(전체/업무/개인/공부) 클릭 → 해당 카테고리만 표시. 각 탭에 완료/전체 개수가 함께 표시됩니다
- **진행률**: 헤더에 전체 완료/전체 개수와 퍼센트 진행률 바가 표시됩니다
- **도움말**: 헤더의 "?" 아이콘을 누르면 이 문서(README)가 새 탭에서 열립니다

## 데이터 저장 및 백업

- 모든 데이터는 브라우저의 `localStorage`에 저장됩니다. **기기·브라우저별로 별도 저장되며 서로 동기화되지 않습니다.**
- 화면 하단 **"내보내기"** 버튼으로 현재 목록을 JSON 파일로 다운로드해 백업할 수 있습니다.
- **"가져오기"** 버튼으로 백업 JSON 파일을 선택하면 현재 목록을 그 내용으로 덮어씁니다(확인 창이 뜹니다).
- 브라우저의 사이트 데이터/캐시를 지우면 로컬에 저장된 할 일도 함께 사라지므로, 주기적으로 내보내기를 해서 백업해두는 걸 권장합니다.

## 모바일에서 설치

1. https://ehdqnddl.github.io/todoapp/ 접속
2. iOS Safari: 공유 버튼 → "홈 화면에 추가" / Android Chrome: 메뉴 → "앱 설치"
3. 홈 화면 아이콘으로 실행하면 오프라인 상태에서도 캐시된 화면이 열립니다.

## 로컬에서 실행 (개발용)

```
Todoapp\run.bat
```

Python이 설치되어 있으면 `http://localhost:8000`에서 서버가 뜨고 브라우저가 자동으로 열립니다.

## 테스트

```
npm run test:unit   # 데이터 로직 단위/시나리오 테스트 (Node 내장 테스트 러너)
npm run test:e2e    # 실브라우저 시뮬레이션 테스트 (Playwright)
npm run test:all    # 위 둘 다
```

테스트 계층 구성, 새 기능 추가 시 어떤 테스트를 어디에 추가해야 하는지는 [TESTING.md](TESTING.md)에 정리되어 있습니다.

## 기술 스택

바닐라 JavaScript(ES Modules), HTML, CSS — 프레임워크·빌드 도구 없음. `manifest.json` + `service-worker.js`로 오프라인 지원(PWA).
